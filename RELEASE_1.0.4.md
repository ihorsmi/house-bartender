# House Bartender 1.0.4

Release date: 2026-03-17

## Highlights

- Fixed cocktail recipe-library view switching in bartender and admin-visible catalog screens.
- Replaced the single toggle button with explicit `Line view` and `Grid view` controls.
- Kept line view as the default catalog layout while making the active state much clearer.

## Changes

### Fixed

- Fixed recipe library line/grid switching so the selected view applies reliably.
- Fixed the bartender cocktail library view control.
- Fixed the admin-accessible bartender cocktail library view control.

### Changed

- Replaced the single flip button with a grouped two-button view selector.
- Added active button styling based on the saved cocktail catalog view mode.
- Applied the same explicit line/grid selector pattern to the user cocktail catalog for consistency.
